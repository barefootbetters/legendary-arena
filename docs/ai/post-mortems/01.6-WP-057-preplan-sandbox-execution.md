## Section 8 — Reality-Reconciliation Findings

**One minor finding.** No blocking issues. No post-mortem fixes required.

### Finding 8.1 — TypeScript strict-mode narrowing required a first-compile fix

**What the session prompt said:** The Task 3 / Task 5 skeletons use the
idiom `const drawnCard = deck[0];` and Fisher-Yates's `const tmp =
shuffled[i]; shuffled[i] = shuffled[j]; shuffled[j] = tmp;` directly.

**What the compiler required:** The WP-056-inherited `tsconfig.json` at
`packages/preplan/tsconfig.json:9-10` sets `noUncheckedIndexedAccess: true`
and `exactOptionalPropertyTypes: true`. Under these settings:

- Array indexed access returns `T | undefined` — so `deck[0]` is `string
  | undefined` and cannot be assigned to a `string` target without
  narrowing.
- Optional properties (`invalidationReason?:`) forbid explicit `undefined`
  assignment — the field must be omitted from the object literal, not set
  to `undefined`.

**Resolution (first-compile, not post-mortem):** Replaced the
`deck.length === 0` guard + `deck[0]` pattern with a destructured
`drawnCard === undefined` guard that narrows the type in the same
statement. Fisher-Yates's swap uses explicit `as T` assertions since
indices `i` and `j` are guaranteed valid by the loop construction.
`createPrePlan` omits `invalidationReason` from its return literal (the
field is optional). Test helper `buildActivePlan` similarly omits the
field from its default.

**Classification:** Session-prompt skeleton-vs-compiler divergence, not a
spec ambiguity. The post-resolution implementation preserves every locked
semantic (null-on-inactive, empty-deck null-return, revision increment,
spread-copy discipline) and passes all 24 verification steps. No
architectural or scope implication. WP-058 should inherit the narrowing
pattern without ceremony.

**Follow-up:** None required. The skeletons in session prompts for future
`preplan`-category WPs should use the destructured guard + `as T` swap
pattern as a matter of course; the next author will reach the same fix
without guidance.
